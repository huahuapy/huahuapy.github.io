hexo+github搭建个人博客

发表于 2020-12-24 | 分类于 个人博客 |

​ 在学习过程中，通过博客记录个人学习所得以及心得是一种可以加深理解、促进分享的好习惯。本人选择的是通过github+hexo搭建个人博客的方法。现在就记录下我的搭建过程吧。

简要介绍Hexo

​ Hexo是一款基于Node.js的静态博客框架，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub和Heroku上，是搭建博客的首选框架。Hexo同时也是GitHub上的开源项目，参见：hexojs/hexo 如果想要更加全面的了解Hexo，可以到其官网 Hexo 了解更多的细节。

​ 其实在很多时候，当想到在github上搭建个人网站时已经在github上玩转一段时间了（最起码有了github账号了。。）。所以本篇博客并不会讲github注册、下载Node.js等，这些也可以在网站找到各式各样五花八门的教程，就从Hexo开始。

Github创建个人仓库

​ Github中创建仓库很简单，就是点击New repository创建新仓库， 但是创建个人博客仓库的仓库名必须要注意一下啊：仓库名应该为 用户名.github.io。这是固定写法。

Typora编辑器编辑并发布文章

​ 在新增文章时已经发现hexo新增的文件为md格式，这是Markdown的文件格式，也需要Markdown文件编辑器。市面上有很多Markdown文件编辑器，个人选择有不同，本人选择的是Typora。

​ Typora的下载和安装可以登录Typora官网。Typora安装完成后打开，可以直接点击文件—>打开文件夹，然后选择myblog所在文件夹，就可以在source->_posts中找到所需编辑文章。

编辑好文章后保存，就可以通过hexo genereste \ hexo server先在本地预览一下效果，再通过hexo deploy部署到github上。
