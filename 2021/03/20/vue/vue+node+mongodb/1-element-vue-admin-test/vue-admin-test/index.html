  • 打开vscode，载入项目文件。与此同时，在项目文件夹中打开cmd， 输入npm run serve启动项目。

2. 后端搭建(node+express+mongodb)

  • 由于这只是一个简略的test项目，所以后端就不单独新建文件夹，在项目文件夹中新建了SERVER文件夹。
  • 直接在项目文件夹中开新终端， npm i express@next mongoose cors
  • SERVER文件夹新建index.js.
  • index.js代码如下：

  • 建立数据库（index.js中直接写即可，项目练手的而已，很简单，不分开了）

    这中新建方式只要电脑中装了mongodb就可以使用，mongodb://localhost:27017是本机mongodb默认的连接端口，vue-admin-test是自己新建的数据据库名称，后续{}中的参数要写以下，不然后端容易报很多数据库错误。

    然后使用new mongoose.Schema对数据库中字段进行描述，这里没有很多字段，就是文章的标题与内容，建好后就可以使用Article进行操作了

  • 启动后端 cmd -> nodemon SERVER/index.js
